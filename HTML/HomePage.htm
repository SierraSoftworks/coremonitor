CoreMonitor

HTTP API Documentation

CoreMonitor provides a web based API for controlling notifications displayed on the device. This allows developers to easily control CoreMonitor using nothing more than the user's web browser. The API makes use of HTTP GET requests to a specific URL which are then used to trigger the functions of the device. CoreMonitor also provides an XML based commands API which can be used for more advanced functions like registering menu items etc.
