CoreMonitor

Notifications API

Part of the CoreMonitor WebAPI is the Information interface which provides information about an existing CoreMonitor instance to the device through web requests.
Web requests are made using either HTTP GET or HTTP POST requests to the /system/* pages
The following are the currently available Information pages
  • http://localhost:56302/system/version
    Returns the current CoreMonitor version running on the system
  • http://localhost:56302/system/version/show
    Shows the current CoreMonitor version running on the system as a notification on the device
